# **PDE Project**

## To compile and run the project : 

```bash
g++ Project\*.cpp -o project.exe
         .\project.exe  
```

To generate the graph: 

```bash
python plot_convergence.py
```

## Results 

The matrix invertibility test is implemented at the end of Matrix.cpp and is 
executed when running the project.

The numerical results are in Convergence_results.csv.
The convergence plot is generated in Convergence.png using the Python script.
